# Strascook

Application web de réservation pour chef privé — permet aux clients de consulter les menus, composer un panier et réserver un événement. Espace backoffice pour la gestion des réservations, menus et disponibilités.

## Fait par

Kuider Boutaiba — projet d'équipe (Wild Code School)

## Fonctionnalités

- Consultation et filtrage des menus (végétarien, vegan, sans gluten, sans lactose)
- Système de panier persistant
- Réservation avec vérification des disponibilités en temps réel
- Backoffice : gestion des réservations, menus et calendrier
- Authentification avec rôles (admin / chef)

## Technologies

- React · TypeScript · Vite
- Material UI (MUI)
- React Router v7
- Biome (linter/formatter)

## Lancer le projet

```bash
npm install
npm run dev
```

L'application sera accessible sur `http://localhost:5173`.

## Scripts

| Commande | Description |
| :--- | :--- |
| `npm run dev` | Serveur de développement |
| `npm run build` | Build de production |
| `npm run lint` | Vérification du code |
| `npm run lint:fix` | Correction automatique |
